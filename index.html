Javascript / WebDev things you should know

ES6 / ECMAScript 2015

Brought us a lot of new syntax, like the fat arrow function


// ES5
function add(x, y) {
    return x + y;
}

// ES6
(x, y) => x + y;

Transpiling

Converting ES6 to ES5, or TypeScript to Javascript

TypeScript

A language developed by Microsoft. It is a superset of Javascript which gives us static typing.

JSX

Javascript and XML. A react-specific grammar extension to Javascript.

Webpack

The module bundler for Javascript applications
